# Preprocesamiento de datos para análisis de Riesgo de Crédito

Se realizó el preprocesamiento de un conjunto de datos 466,285 clientes de Lending Club entre 2007 y 2014. El objetivo es preparar los datos para su análisis eliminando valores nulos, convirtiendo variables categóricas a variables dummy y transformando fechas y variables numéricas para su posterior uso en un modelo de riesgo.
